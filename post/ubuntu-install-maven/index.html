Ubuntu 安装 Maven，解决 IDEA Unable to import maven project

2020-03-07

Contents

  • 首先你需要
  • 安装 maven
    • apt 安装
    • maven 官网下载安装
  • bug
  • 参考

首先你需要

  • JDK 环境变量

安装 maven

以下两种方式二选一

apt 安装

1
2

sudo apt update
sudo apt install maven

不需要配置环境，执行 mvn -version ，显示版本信息的话就是安装成功

maven 官网下载安装

maven 官方下载

全新版本是3.6.3，下载 apache-maven-3.6.3-bin.tar.gz ，copy 到 /opt 下解压

1

sudo tar -zxv -f apache-maven-3.6.3-bin.tar.gz

创建软链接

1

sudo ln -s /opt/apache-maven-3.6.3  /opt/maven

配置环境变量 不建议直接在 profile 上修改

1

sudo nano /etc/profile.d/maven.sh

我不会vim 复制到里面，修改你的JDK路径

1
2
3
4

export JAVA_HOME = 你的JDK路径
export M2_HOME = /opt/maven
export MAVEN_HOME = /opt/maven
export PATH = ${M2_HOME}/bin:${PATH}

给 maven.sh 添加执行权限

1

sudo chmod 755 maven.sh

使配置生效

1

source maven.sh

终端输入 mvn -version 验证

如果你安装失败了，记得删除 maven.sh 文件，因为你的 maven.sh 可能有错误，profile 环境变量错误会导致桌面循环登陆

bug

修改之前的项目，把 Maven home directory 修改为 /opt/maven，删除项目 .idea 重新导入，没报错，启动 tomcat 全部类爆红，连 springframework 包都报错，看起来是依赖全部都没加载，IDEA 提示 Unable to import maven project 错误，但是 pom.xml 并没有报错 电脑太卡，过了两分钟才报错

一顿搜索后，好像是 maven3.6.3 和 IDEA2019.1.4 冲突了

IDEA2019.1.4 自带的 maven 插件版本就是 3.6.1 的,那就重新下载 3.6.1 的试一下

maven 归档 下载 3.6.1，按照上面的操作再来一次，没报错，运行成功 naisuuuuuuuuuuuu
