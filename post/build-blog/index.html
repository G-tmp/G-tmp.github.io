Ubuntu 用 Hugo + Github 搭建个人博客

2020-02-17

Contents

  • 首先你需要
  • 第1步：安装 Hugo
  • 第2步：创建 Hugo 项目
  • 第3步：选择一个 Hugo 主题
  • 第4步：部署到 Github
  • 参考

首先你需要

  • Github 账号

  • Git 环境

第1步：安装 Hugo

Hugo 版本不要选择太低的，我之前已经下载了一个 0.3 版本的，下载好主题，运行的时候提示该主题要求 Hugo 不低于 0.5 。 卸载重装一条龙

Github 下载 Hugo

我的系统是Ubuntu18.04 64位，选择 hugo_0.64.0_Linux-64bit.deb 下载。

1
2

#安装
sudo dpkg -i hugo_0.64.0_Linux-64bit.deb

终端运行：

1
2

#测试
hugo version

提示 Hugo Static Site Generator 开头的信息 ，安装成功

第2步：创建 Hugo 项目

任意选择一个目录，我是在 home 目录下创建的

1

hugo new site Blog  		#你可以把 Blog 改为任意名称

提示 Congratulations! 创建成功

第3步：选择一个 Hugo 主题

主题市场 随便选

我选择 even 主题 ，简洁

下载：

1
2

cd Blog
git clone https://github.com/olOwOlo/hugo-theme-even themes/even

Blog/themes 目录下多了一个 even 文件夹

进入 ~/Blog/themes/even/exampleSite 目录，复制并覆盖 ~/Blog 目录下的 config.toml 文件和 content 文件夹。

1
2
3

cd ~/Blog
cd themes/even/exampleSite/
cp -r * ~/Blog

你可以修改 ~/Blog/config.toml 来自定义网站样式

启动：

1
2

cd ~/Blog
hugo server

提示 Web Server is available at http://localhost:1313/ (bind address 127.0.0.1) 成功

浏览器打开 http://localhost:1313/ 会展示主题作者的文章样例，你可以在 ~/Blog/content/post 下编辑，以后写的博客也是放在这个目录下 ，把文章样例的 draft 改为 true 就不显示了。

你可以在 Blog 目录下执行 hugo new post/XD.md ， 这个命令会在 Blog/content/post 下生成附带Markdown格式的markdown文本，touch 只是创建一个白板

第4步：部署到 Github

登陆 Github，点击 New repository ，在 Repository name 这一栏输入 <你的 Github 用户名>.github.io ，点击 Create repository

打包：

1
2

cd ~/Blog
hugo

执行后会在 Blog 下多出一个 public 目录，你只需要把这个目录 push 到 Github

推送到你刚才创建的 Github 库：

1
2
3
4
5
6

cd ~/Blog/public
git init
git remote add origin https://github.com/<Github 用户名>/<Github 用户名>.github.io.git
git add .		#有个点号
git commit -m "XD"
git push -u origin master

推送到Github 后过一会就能访问拉 ， <你的 Github 用户名>.github.io
